Doubledown

Sync local changes to a remote directory

Installation

Mac OS X

Download doubledown-0.0.2.tar.gz.  Unpack it and run the installer package it contains.

Debian/Ubuntu Linux

Download doubledown_0.0.2_all.deb.  Run dpkg -i doubledown_0.0.2_all.deb as root.
